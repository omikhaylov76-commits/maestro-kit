# library — пополняемые полки набора

Сюда складывается то, что переезжает из проекта в проект. Подключение в конкретный проект:
`npx degit omikhaylov76-commits/maestro-kit/library/<полка> library/<полка>`

- `design/` — дизайн-токены и правила красоты (конечный словарь!).
- `skills/` — заметки, какие скиллы под какие проекты брать.
- `checklists/` — чек-листы аудитов, которые использует /build.

Правило полок: кладём только то, чем реально пользовались ≥2 раз. Иначе это шкаф ненужных вещей.
