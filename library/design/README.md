# design — дизайн-токены

Правило конечного словаря (анти-разползание, привет gbrain): в проекте живёт МАЛЫЙ набор
токенов — фон/текст/акцент/статусы, 1–2 шрифта, одна шкала отступов (например 4/8/16/24/32).
Новый цвет или шрифт — только через осознанное решение, а не «ещё один оттенок по вкусу».

Сюда кладутся наборы токенов владельца (например: yahtzee / коінбайс / слаш — theme.css,
variables.css, tokens.json, DESIGN.md). Дирижёр на фичах внешнего вида берёт токены отсюда,
а не выдумывает свои.
