# Changelog

## 1.0.0 — 2026-07-10
Первый выпуск. Гибрид vibe-build-kit + LLM-вики Карпати:
- /build: память в wiki/, шаг G-WIKI в цикле фичи, вики-линт на границе фазы,
  токен-бюджет ШАГА 0 (вход через 4 файла), правило коротких порций.
- Новые команды: /status, /wiki (запомни/найди, накопление ответов), /handoff (автопередачка).
- Анти-разрастание: правило одного места, лимиты страниц, Icebox в roadmap,
  чердак wiki/attic/, раздел «чего сознательно НЕТ» в spec.
- code-reviewer без хардкода main (ревью против merge-base текущей ветки).
- library/: полки design-токенов, skills, чек-листов.

## 1.1.0 — 2026-07-10
- Новый ШАГ 3.7 «Глубокая проработка швов»: каталог швов/контрактов, сквозные
  трассировки сценариев, отказы на швах, враждебный само-разбор — до аудита и до кода.
  Масштаб по проекту (лёгкий проход для простых). Родился из практики (Citadel MFC-000):
  задать вектор глубоко в начале, чтобы меньше тёмных пятен потом.

## 1.2.0 — 2026-07-10
- Веточная дисциплина в цикл фичи: код — на ветке `task/<slug>` (зелёные тесты + code-reviewer
  + ⛔ → слияние в main), доки/chore — прямо в main. Масштаб-осознанно (простой проект без веток ок).
- ШАГ 3.7: строка коэрентности — сверить README/CLAUDE.md/spec с углублённым дизайном до кода
  (ловит устаревшие утверждения, особенно про секреты). Оба урока — из обкатки на Citadel.
